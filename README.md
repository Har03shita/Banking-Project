# Bank Management System (Full Stack)

A full-stack banking application built using **Spring Boot**, **Spring Security**, **MySQL**, and **React.js**.

---

## Tech Stack

- Java (Core Java, Java 8+)
- Spring Boot
- Spring Security (RBAC)
- RESTful APIs
- MySQL Database
- React.js Frontend

---

## Features

- User Authentication & Authorization
- Role-Based Access Control (RBAC)
- Deposits and Withdrawals
- Fund Transfers
- Transaction History
- Responsive React UI with real-time updates
